# Notes

## Containers

These are required when using grid system.

### Responsive
```html
<div class="container">
    <!-- Content here... -->
</div>
```

### Fixed-Width 
`max-width` changes at each breakpoint.


### Fuild-Width
`100%` wide all the time.


```html
<div class="container-fluid">
    <!-- Spans entire width of viewport -->
</div>
```
